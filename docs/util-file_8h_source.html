122/**
123 * \brief Open a new File
124 *
125 * \param ffc flow container
126 * \param sbcfg buffer config
127 * \param name filename character array
128 * \param name_len filename len
129 * \param data initial data
130 * \param data_len initial data len
131 * \param flags open flags
132 *
133 * \retval ff flowfile object
134 *
135 * \note filename is not a string, so it's not nul terminated.
136 *
137 * If flags contains the FILE_USE_DETECT bit, the pruning code will

140 * properly updated.
