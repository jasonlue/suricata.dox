Detailed Description

Author
Pablo Rincon Crespo pablo.nosp@m..rin.nosp@m.con.c.nosp@m.resp.nosp@m.o@gma.nosp@m.il.c.nosp@m.om

Bs2Bm use a simple context array to determine the charactes that are not present on the pattern. This way on partial matches broken by a char not present, we can skip to the next character making less checks
